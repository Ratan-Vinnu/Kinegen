# Examples

This directory contains example robots and demonstration scripts for Kinegen.

The examples are intended to:

- Demonstrate the library API
- Verify mathematical correctness
- Provide reference implementations for users
- Showcase examples of supported robot configurations

## Contents

Current examples include:

- '2R_Robot.json' - Example robot description
- `3R_Robot.json` - Example robot description
- `MR_LP_6R_Robot_Arm.json` – Forward kinematics validation against a textbook example

More examples will be added as new functionality is implemented.

Planned examples include:

- SCARA robot
- UR5 forward kinematics
- Jacobian computation
- Inverse kinematics
- Visualization demos
